
# 正则表达式

| 符号        | 作用                                    |
| ------------|:---------------------------------------:|
| [0-9] \d    | 全部数字                                |
| \w          | 单词类字符 a-z A-Z 0-9 _                |
| \w          | 非单词类字符                            |
| {2}  {n}    | 前面的表达式匹配n次                     |
| {0,2} {m,n} | 前面的表达式匹配m到n次                  |
| +           | 前面的表达式，出现1到无限次  最少出现1次|
| ?           | 前面的表达式，出现0到1次  最多出现1次   |
| *           | 前面的表达式，出现0到无限次             |
