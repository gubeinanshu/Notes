
（.）在发送的时候就是嘀的声音
（-）在发送的时候就是嗒的声音
嘀是一个时间长度
嗒是三个时间长度
发送的时候，点和划之间停一个时间点长度
字符和字符之间停留3个时间点长度
单词和单词之间停留7个时间点长度
